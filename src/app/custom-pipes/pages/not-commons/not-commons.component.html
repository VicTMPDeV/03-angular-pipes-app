Slice Pipe (Cortar elementos en Arrays y Strings)

Incluídos en Angular (Usualmente en el CommonModule de @angular/common).

Original

{{heroes}}

slice : 0 (posicion desde INCLUIDO) : N (posición hasta EXCLUIDO)

{{heroes | slice:0}}

slice : 0 : 2

{{heroes | slice:0:2}}

slice : 1 : 3

{{heroes | slice:1:3}}

slice : 3 : (heroes.length)

{{heroes | slice:0:(heroes.length)}}

Key-Value Pipe (Recorrer Objetos Literales JS)

Incluídos en Angular (Usualmente en el CommonModule de @angular/common).

Si intentamos recorrer un objeto literal JS con un ngFor, saltará un error porque no está preparado para ello.

Con el uso del keyvalue pipe, sí que podemos recorrer el objeto literal con ngFor
  • {{item.key}} : {{item.value}}
